Wikitongues Phrasebooks, V0
==============

Requires [Grunt CLI](https://www.npmjs.com/package/grunt-cli)

Make sure to run `npm install` and `bower install`

Start project with:

    $ grunt

Go to [http://localhost:3000](http://localhost:3000)
